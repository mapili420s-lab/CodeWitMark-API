# CodeWithMark - Spring Boot Project

## Overview
CodeWithMark is a Spring Boot backend application created to practice and demonstrate REST API development, backend architecture, and Java programming fundamentals.

This project represents my journey as an aspiring web and backend developer.

---

## Features
- REST API development using Spring Boot
- Basic backend structure and organization
- CRUD-style operations (if applicable in your project)
- Simple and clean project architecture
- Practice for real-world backend development

---

## Tech Stack
- Java
- Spring Boot
- Maven
- REST API
- Git & GitHub

---

## How to Run

1. Clone the repository: git clone https://github.com/mapili420s-lab/CodeWitMark-API

2. Open the project in your IDE (IntelliJ IDEA or VS Code)

3. Build the project using Maven

4. Run the application: ./mvnw spring-boot:run for macOs, mvn spring-boot:run for Windows

5. Access the application (if API is running):http://localhost:8080
